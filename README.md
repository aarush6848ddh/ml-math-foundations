# ML Math Foundations

A personal study repo documenting my journey learning the mathematics behind machine learning and AI. Each topic includes working Python code and a wiki page that explains the concept in plain English, shows the key formulas, and connects it directly to how modern AI systems work.

## Why I made this

I kept running into the same wall when reading ML papers and working through deep learning projects: the math was blocking me. Terms like eigenvectors, gradients, and probability distributions would show up and I would have a surface-level understanding at best. I decided to go back to basics and actually learn the math properly, starting with linear algebra.

I'm following 3Blue1Brown's website as my primary guide because it builds geometric intuition first before introducing the notation. This repo is how I take notes: code I can run, formulas I can reference, and explanations written in my own words.

The wiki is also something I can share. If you're learning ML and hitting the same walls, hopefully this is useful.


Each topic lives in two places. The Python file in the subject folder is where I write and run the actual code while learning. The corresponding wiki page is the write-up: concept explained in plain English, LaTeX formulas rendered with MathJax, the code from the Python file, and a section on how it connects to AI.
