Inverse Matrices, Column Space, and Null Space

From 3Blue1Brown, Essence of Linear Algebra, Chapter 7
Linear Algebra Lesson 07
Concept

Undoing a Transformation

Every linear transformation is a function that takes a vector and produces a new one. The inverse matrix is simply the function that undoes that transformation. If matrix A rotates space 90 degrees clockwise, then A inverse rotates it 90 degrees counterclockwise, putting everything back where it started.

The core idea: The inverse of A, written \(A^{-1}\), satisfies \(A^{-1}A = I\), where \(I\) is the identity matrix. Applying the transformation and then its inverse leaves every vector unchanged.

Solving Ax = b

The most useful application of the inverse is solving the system \(Ax = b\). You have a transformation A, a known output vector b, and you want to find which input vector x produced it. If A is invertible, the answer is \(x = A^{-1}b\). You are literally tracing the transformation backwards.

When the inverse does not exist

If the determinant of A is zero, the transformation squishes space into a lower dimension. You cannot invert a squish: many different input vectors all land on the same output, so there is no way to know which one you started from. The matrix is called singular, and NumPy will raise a LinAlgError if you try to invert it.

Column space and rank

The column space of a matrix is the set of all possible output vectors. It is the span of the matrix's columns. A full-rank 2x2 matrix has column space equal to all of 2D space. A rank-deficient matrix's column space is just a line through the origin. Rank is the dimension of that column space.

Null space

The null space is the set of all input vectors that the transformation maps to the zero vector. For a full-rank matrix, only the zero vector itself maps to zero. For a rank-deficient matrix, an entire line (or plane) of vectors all collapse to the origin. These vectors are invisible to the transformation.

Formulas

Key Formulas

Inverse definition
\[ A^{-1}A = AA^{-1} = I \]
Solving a linear system
\[ Ax = b \implies x = A^{-1}b \]
2x2 inverse formula
\[ \begin{bmatrix} a & b \\ c & d \end{bmatrix}^{-1} = \frac{1}{ad - bc} \begin{bmatrix} d & -b \\ -c & a \end{bmatrix} \]
Null space definition
\[ \text{null}(A) = \{ x \mid Ax = \mathbf{0} \} \]
Code

Code Examples

Part 1: Solving a linear system

Compute the inverse of A and multiply it by b to find x. Then verify that A @ x gives back b.

Python lesson-07-inverse-column-null.py 
import numpy as np

matrix_a = np.array([
    [1, 4],
    [3, 5]
])

vector_b = np.array([1, 7])

a_inv = np.linalg.inv(matrix_a)
x = np.matmul(a_inv, vector_b)
verification = np.matmul(matrix_a, x)

print(f"Solution x: {x}")
print(f"Verification A @ x: {verification}")
print(f"Matches b: {np.allclose(verification, vector_b)}")
Output
Solution x: [ 3.28571429 -0.57142857]
Verification A @ x: [1. 7.]
Matches b: True

Part 2: Column space and rank

A full-rank matrix spans all of 2D space. A rank-deficient matrix squishes the plane onto a single line. np.linalg.matrix_rank counts the number of linearly independent columns.

Python lesson-07-inverse-column-null.py 
# full rank: columns are linearly independent, spans full 2D space
full_rank = np.array([
    [1, 2],
    [3, 4]
])

# rank deficient: column 2 = 2 * column 1, collapses to a line
rank_deficient = np.array([
    [1, 2],
    [2, 4]
])

rank1 = np.linalg.matrix_rank(full_rank)
rank2 = np.linalg.matrix_rank(rank_deficient)

print(f"Full rank matrix rank: {rank1}")
print(f"Rank deficient matrix rank: {rank2}")
Output
Full rank matrix rank: 2
Rank deficient matrix rank: 1

Part 3: Null space

For the rank-deficient matrix, the vector [2, -1] maps exactly to zero. That is because 2 times column 1 minus 1 times column 2 cancels out. The vector lives in the null space: the transformation makes it invisible.

Python lesson-07-inverse-column-null.py 
null_vector = np.array([2, -1])
result = np.matmul(rank_deficient, null_vector)
print(f"rank_deficient @ [2,-1] = {result}")
Output
rank_deficient @ [2,-1] = [0 0]

Part 4: Irreversibility

Attempting to invert a singular matrix raises a LinAlgError. Once information is destroyed by a rank-deficient transformation, it cannot be recovered.

Python lesson-07-inverse-column-null.py 
singular_matrix = np.array([
    [1, 2],
    [2, 4]
])

try:
    inverse = np.linalg.inv(singular_matrix)
    print(f"Inverse: {inverse}")
except np.linalg.LinAlgError as e:
    print(f"Cannot invert: {e}")
Output
Cannot invert: Singular matrix
AI Connection

How This Connects to AI

Why this matters for machine learning

Solving \(Ax = b\) is exactly what a neural network does during inference. The network learns the matrix A during training: the weight matrix that transforms an input into an output. At inference time it solves for the output given the input, one layer at a time. Understanding this as a linear system makes the math behind forward passes intuitive.

Rank-deficient weight matrices destroy information permanently. Just like a singular matrix has no inverse, a layer with collapsed rank cannot be undone. Data that maps to the null space is gone forever. This is why full-rank weight matrices are critical for network capacity, and why initialization schemes like Xavier and Kaiming init are carefully designed to keep weight matrices well-conditioned from the start.

The null space of a weight matrix represents features the layer ignores completely. If an important input feature happens to lie in the null space, the network can never learn from it, no matter how long you train. This is one reason why architecture design and feature engineering matter even in deep learning.
